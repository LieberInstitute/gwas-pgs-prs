# Plan: PRS Diagnosis Association and Threshold Interpretation

## Summary

- Demographics confirm 119 samples: 39 MDD, 40 Bipolar/BPD, 40 Controls; no SCZ cases.
- Existing work checked GraphPred/+score vs PRSice correlations, but not diagnosis separation.
- Analyze MDD, BPD, and union MDD+BPD against Controls, plus Bipolar vs MDD as a secondary contrast.
- Report all PRSice thresholds and flag internally best thresholds as exploratory because choosing by best separation in the same sample is overfit.

## Implementation

- Create a reproducible R script that joins:
  - `MBv_demographics_n119.tab`
  - `mbv_prs_merged_scores.tsv`
  - `qc/mbv.pca.eigenvec`
- Validate sample joins, duplicate IDs, diagnosis counts, score finiteness, and PC availability.
- Fit unadjusted logistic models: `case_status ~ PRS_z`.
- Fit sensitivity logistic models: `case_status ~ PRS_z + Sex + PC1 + PC2 + PC3 + PC4 + PC5`.
- Standardize each PRS over all 119 samples before modeling.
- Report mean difference, Cohen d, OR per 1 SD PRS, CI, p-value, FDR, AUC, and sample counts.
- Mark best PRSice thresholds by highest AUC within each contrast and score family as `exploratory_best_internal`.

## GWAS Overlap Integration

- Include the existing MDD/BPD significant-variant overlap in the report.
- Interpret overlap cautiously because raw genome-wide significant variant overlap is not equivalent to PRS predictive overlap.
- Note that PRSice thresholds include many sub-significant variants, not only genome-wide significant variants.
